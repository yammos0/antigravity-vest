# Antigravity Unlock / Vesting Front-Run Engine

High-performance AI system for crypto-analytics.
Engineered for institutional-grade prediction of price pressure from token unlocks.

## Architecture

- **Backend**: Python 3.12, FastAPI, Celery, DuckDB
- **Frontend**: Next.js, Tailwind, WebSockets

## Quick Start

### Backend
```bash
cd backend
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
uvicorn app.main:app --reload
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
